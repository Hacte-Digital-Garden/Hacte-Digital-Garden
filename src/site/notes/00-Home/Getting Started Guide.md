## ğŸŒ± What is a Digital Garden?

A digital garden is a living, evolving collection of knowledge that:

- **Grows organically** - content develops over time
- **Connects ideas** - uses links to show relationships between concepts
- **Embraces process** - includes works in progress and evolving thoughts
- **Invites exploration** - encourages non-linear navigation

Unlike a blog (chronological) or wiki (encyclopedic), a digital garden is a space for cultivating ideas.

## âœï¸ Creating New Content

### Page Naming Conventions

- Use descriptive, kebab-case names: `Autoetnografia-Lara.md`
- Avoid spaces and special characters
- Keep names concise but clear

### Page Template

markdown

```markdown
---
tags: [tag1, tag2, tag3]
created: YYYY-MM-DD
updated: YYYY-MM-DD
---

# Page Title

> *Brief description or context*

## Content

[Your main content]

---

## Related
- [[Related-Page-1]]
- [[Related-Page-2]]

---

**Tags:** #tag1 #tag2 #tag3
```

### Linking Best Practices

- **Link generously** - Create connections between related ideas
- **Use descriptive link text** - `[[Autoetnografia-Lara|Lara's reflection on error]]`
- **Link to MOCs** - Help readers find broader context
- **Bi-directional linking** - Link back to pages that link to you
